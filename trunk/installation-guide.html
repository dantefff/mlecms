Setup

1. Create Page structure

Each language have own pages structure:

2. Configure your MleCMS module

Setup Multilang config tab in MleCMS administration area

3. Put tags to your template

Init action on top your template. This tag set new smarty vars - {$lang_parent}, {$lang_locale}


Mle language switch to your template. Setup in Options tab

4. Build your website easily - use MLE Blocks, Mle Snippets and Mle Translator

Mle Blocks, Mle Snippets are similar as Global Content Blocks, but with support MLE. You can create different content for every defined language.

Mle Translator is easy traslate manager. Put to your template {translate text="your multilang string"} and change this key for every language in administration area.

5. Example usage MLE CMS with modules

Use {$lang_parent} and {$lang_locale} variables. Examples:

{menu loadprops=0 template='cssmenu_ulshadow.tpl' childrenof=$lang_parent lang=$lang_locale}
{news number='3' category=$lang_parent lang=$lang_locale} - don't remember create news category
{search search_method="post" lang=$lang_locale} - search don't works separate for every language, now :(
{breadcrumbs starttext='You are here' root='Home' delimiter='»' lang=$lang_locale}
